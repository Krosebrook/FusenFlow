# Quickstart (60 Seconds)

1. **Clone & Install**
   ```bash
   npm install
   ```
2. **Set API Key**
   Ensure your environment has `API_KEY` configured for Google Gemini.
3. **Run Dev**
   ```bash
   npm run dev
   ```
4. **First Interaction**
   - Click "New Document".
   - Toggle "Coach Active" (Magic Mode).
   - Write "This is a good start."
   - Highlight the word "good" and ask the AI to "Make it punchy".
